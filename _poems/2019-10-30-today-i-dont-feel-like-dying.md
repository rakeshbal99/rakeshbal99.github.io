date: 2019-10-30
description: "A triolet about rebirth and resilience."

Today I don't feel like dying  
Today I want to be born again  
Because today I want to see the world crying  
Today I don't feel like dying  
The world that has been so lying  
Where all my efforts go in vain  
Today I don't feel like dying  
Today I want to be born again
